# se-ml.github.io

Software Engineering for Machine Learning Lab Homepage


### Prereq
Install (jekyll and bundler)[https://jekyllrb.com/docs/].

### Development
```
$ git clone https://github.com/SE-ML/se-ml.github.io
$ cd se-ml.github.io
$ bundle install
$ bundle exec jekyll serve
```

### Deployment
```
$ ./bin/deploy se-ml
```
